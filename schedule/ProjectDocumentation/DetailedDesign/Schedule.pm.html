NAME

Schedule - read and write schedule files

VERSION

Version 1.00

SYNOPSIS

    use Schedule::Schedule;
    
    my $sched = Schedule->read('myschedule_file.txt');
    
    my $teachers = $sched->teachers;
    my $courses  = $sched->courses;
    my $conflicts = $sched->conflicts;
    
    $sched->teachers->add(...);
    $sched->course->add(...);
    $sched->calculate_conflicts(...);
    
    $sched->write('my_new_schedule_file.txt');

DESCRIPTION

This module provides the top level class for all of the schedule objects.

The data that creates the schedule can be saved to an external file, or read in from an external file.

This class provides links to all the other classes which are used to create and/or modify course schedules
